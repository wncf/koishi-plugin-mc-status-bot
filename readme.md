# koishi-plugin-mc-status-bot

[![npm](https://img.shields.io/npm/v/koishi-plugin-mc-status-bot?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-mc-status-bot)

轻松获取我的世界服务器信息


指令：mc
可用的选项有：
    --name 服务器名称 (为空获取所有)



可用的子指令有：
    mc set 新增/修改一个服务器信息

​	--name 名称

​	--address 地址:端口 (服务器的ip和端口，端口不填为25565)

   mc del 删除一个服务器
    --name <名称>
   mc list 获取添加过的服务器列表
